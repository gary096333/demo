
# [圖書館理系統]
## 項目介紹
本圖書館管理系統採用了前後端分離的設計。客戶端使用 Vue.js 和 Element UI 框架實現，服務端則使用 Spring Boot 和 MyBatis 實現，數據庫使用 MySQL。最後，項目部署到 AWS 的雲端虛擬機上。
